# VentureTech.no

Norges venture studio — vi bygger teknologiselskaper sammen med gründere.

## Teknologi

- **Next.js 14** (App Router)
- **Tailwind CSS** 
- **Vercel** (deploy)

## Sider

| Side | URL | Beskrivelse |
|------|-----|-------------|
| Forside | `/` | Hero, hva vi gjør, prosess, startups, CTA |
| Prosess | `/prosess` | 5-stegs prosess i detalj + FAQ |
| Startups | `/startups` | Portefølje med 10 startup-konsepter |
| Send inn idé | `/sok` | Søknadsskjema for gründere |
| Om oss | `/om` | Bakgrunn, verdier, team, modellsammenligning |

## Kom i gang

```bash
npm install
npm run dev
```

Åpne [http://localhost:3000](http://localhost:3000).

## Deploy til Vercel

1. Push til GitHub via GitHub Desktop
2. Koble repoet til Vercel
3. Vercel bygger og deployer automatisk ved push til `main`
4. Koble domenet `venturetech.no` i Vercel dashboard
